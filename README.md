
# Fractol [Computer graphics project]

Fractol is small fractal exploration program, offeres multiple generated fractals with the ability to tweak parameters, coded with in C using mlx library.

## Fractals available:

- Julia set
- Mandelbrot set 
- Sirpinski triangle
- Tree
- Burningship
- Barnsley fern
- Tricon
